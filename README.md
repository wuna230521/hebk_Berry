## 使用教程：
    前往软件官网 skzs.pro 下载最新版刷课助手
    本Python项目跨平台支持安卓手机、苹果电脑、Windows电脑、iPad、Linux。

    下载 skzs-bate.exe 程序，设置 host（网课登录地址）、user（账号）、pwd（密码）以及 threads（线程数，默认为1，超过1进入奔放模式，建议不超过20）。
